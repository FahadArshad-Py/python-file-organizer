Python File Organizer Automation

This project is a Python automation script that organizes files into folders based on their file extensions.

Features
- Automatically detects file types
- Creates folders dynamically if they do not exist
- Organizes files into categories such as Images, PDFs, Documents, and Others
- Simple and beginner-friendly implementation

Technologies Used
- Python
- os module
- shutil module

Use Case
This script can be used to clean and organize messy directories such as Downloads or project folders.

How to Run
1. Place the script in the target directory
2. Update the `base_folder` variable
3. Run the script using Python
